# Fredrick Portfolio — VS Code Ready


## Run locally

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run preview
```

## Notes

- Uses Vite + React + Tailwind CSS.
- Social icons use `react-icons` so GitHub / LinkedIn / Instagram compile cleanly.
- Design tokens are in `src/styles.css`.
- Components are split into `src/components/site`.
